# $\alpha_n = \frac{2n + 1}{n ^4}$
# $\lim_{x\to\infty} \alpha_n = \frac{2n + 1}{n ^4} = \lim_{x\to\infty} \alpha_n = \lim_{x\to\infty} \frac{2n}{n ^4} = \lim_{x\to\infty} \frac{1}{n^2} = \frac{1}{\infty} = 0 $


# $ | \frac{2n + 1}{n ^4} - 0 | = \frac{2n + 1}{n ^4} < \frac{2n + n}{n ^4} = \frac{3n}{n ^4} = \frac{3}{n^3} = 3 . \frac{1}{n^3} $
